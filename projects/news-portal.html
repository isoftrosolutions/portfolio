News Portal

A dynamic news portal built with PHP and MySQL featuring categorized news articles, search, admin content management, and responsive design for readers across Nepal.

Client: iSoftro Solutions

Project Overview

The News Portal is a fully-functional content publishing platform designed for digital newsrooms. It enables editors to publish, categorize, and manage news articles through an intuitive admin panel, while readers enjoy a fast, responsive frontend optimized for both desktop and mobile consumption.

Built with a clean separation between the admin backend and public-facing interface, the system supports multi-category news management, full-text search, and an article publishing workflow that includes image management and SEO-friendly URL structures.

Features

Category Management

Organize news into hierarchical categories. Manage multiple sections such as politics, sports, technology, entertainment, and more.

Article Publishing

Rich article editor with title, content, featured image, author attribution, publish/draft status, and scheduled publishing.

Search

Full-text search across all articles with keyword matching, category filtering, and sorted results by date or relevance.

Admin Dashboard

Centralized admin panel for managing articles, categories, users, and site settings. Includes analytics and moderation tools.

Responsive Design

Mobile-first responsive layout that adapts seamlessly across phones, tablets, and desktops for an optimal reading experience.

SEO-Friendly URLs

Clean, human-readable article URLs optimized for search engines with customizable URL slugs and meta tag support.

Tech Stack

PHP & MySQL JavaScript CSS
